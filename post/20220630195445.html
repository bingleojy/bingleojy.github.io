Emacs编辑器

2022-06-30 Binglin Li

Emacs是一个可扩展,可定制,可实时自动化显示编辑器

连续按ctrl x m（显示为C-x m）可以打开Emacs帮助文件
选项前的 – 也可以简化为 - 是一样的效果

Emacs用法

$ emacs.exe [--选项 文件名]...

–batch

功能：不做交互式显示，隐含-q选项

$ emacs --batch

–chdir

功能：改变启动目路

$ cd $HOME
$ emacs --chdir demos```

–daemon, --bg-daemon[=NAME]

功能：打开一个以NAME命名的隐藏守护进程（我经常使用此选项）

$ emacs --daemon

–fg-daemon[=NAME]

功能：打开一个以NAME命名的显示守护进程

$ emacs --fg-daemon

–debug-init

打开Emacs lisp调试

$ emacs --debug-init

–display, -d DISPLAY

功能：use X server DISPLAY

$ emacs --display
